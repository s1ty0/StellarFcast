# 虚拟环境的配置

我们根据执行环境的不同，配置了两套虚拟环境:
一套用于时序基础模型（具体见：TS-LibProejct文件夹，注意，这里的python需用到3.8版本）的复现，
一套用于时序大模型（具体见：LLMProject文件夹，此处的python无限制，可以只用最新版本的）的复现。
我们推荐使用conda来管理虚拟环境，这样会方便很多。具体conda的使用方法请自行搜索。
